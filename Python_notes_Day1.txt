Interpreter
1.Translate the program one line of the program at a time into machine code.
2.Stops at the first error.Easier to debug
3.Overall execution time is slower
4.No Intermediate Object code is generated.Efficient memory management.
Ex.Python,Ruby,JavaScript

Compiler
1.Translate the program whole of the program into machine language at once.
2.All error messages are displayed at the end
3.Overall execution time is Faster
4.Intermediate Object code is generated.Extra space on the machine.
Ex:C,C++
